# WimStore

本项目服务器端是以本地APP的形式进行模拟

so:

	1.运行项目前需要先安装服务器端APK（WebSerser.apk）
	2.将WebInfos文件夹拷入手机SD卡根目录中，该文件夹主要存储服务器端数据
	3.导入项目并运行即可
